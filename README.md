# EasyCUST
长春理工大学教务管理、网费充值等便捷管理工具
##使用注意
注意：

* 千万千万！千万别在浏览器界面开窗口，关闭窗口，这将导致软件崩溃！
* 软件的V1.4.2及以后的版本由于学校封堵漏洞已停止开发
* 选课期间教务管理系统不稳定，请不要频繁点击按钮
* 有任何软件使用问题，请使用反馈或联系作者，QQ:767436053
* 提示脚本错误，无视，继续运行即可
* 网费充值需要在学号和密码的输入框里填教务的用户名和密码
* 浏览器模式为所有链接在浏览器中打开，软件默认置顶
* 若登录后还提示未登录，重启软件
